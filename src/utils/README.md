# Utils Module

Reusable utility functions used across the application.

## Contents

- `levenshtein.ts` - String distance calculation for fuzzy matching
- `tfidf.ts` - TF-IDF algorithms for relevance scoring
- `dateUtils.ts` - Date range parsing and validation
- `validationUtils.ts` - Data validation helpers
- `pathUtils.ts` - File path management and backward compatibility

## Characteristics

All utilities are:
- **Pure functions** - No side effects
- **Well-tested** - 100% code coverage
- **Documented** - Clear JSDoc with examples
- **Independent** - No circular dependencies
